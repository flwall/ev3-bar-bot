# EV3 Bar Bot  
Ein autonomer LEGO EV3 Roboter transportiert ein Glas oder Becher in einer mechanischen Halterung („Schiebegabel“).
Der Roboter fährt selbstständig zu einer Getränkeabfüllstation, löst dort über einen Taster oder Sensor den Füllvorgang aus, und transportiert anschließend das gefüllte Getränk zurück.
Ziel ist es, den Roboter so schnell wie möglich zu beschleunigen, ohne dass Getränk verschüttet wird.
Messdaten wie Beschleunigung, Neigungswinkel, Position und Fahrdauer werden in MATLAB erfasst, visualisiert und über eine GUI konfigurierbar gemacht.

Das Projekt verbindet Mechanik, Sensorik, Regelungstechnik, Optimierung und MATLAB-App-Entwicklung.
